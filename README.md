# Me2U

## Overview

**Me2U** is a social networking mobile application that allows users to connect, share posts, chat, and interact with others.

The application focuses on core social interactions such as posting content, liking posts, commenting, and chatting with other users.

Currently, the project is being developed as a prototype using local storage before integrating a backend service.


## Features

### Implemented Features

* User registration
* User login authentication
* Navigation between screens
* User profile (currently using dummy data)
* Create and view posts
* Like posts
* Comment on posts
* Chat system (currently using dummy data)
* Notification system (dummy data)

### Upcoming Features

* Follow users
* Search for users
* Real-time chat system
* Edit profile with persistent data
* Backend API integration
* Database integration
* Push notifications

## Installation

1. Clone the repository

git clone https://github.com/techift/me2u.git


2. Navigate to the project directory

cd me2u

3. Install dependencies

flutter pub get


4. Run the application

flutter run
